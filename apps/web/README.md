# Web App (Angular) Scaffold

This folder intentionally contains a lightweight scaffold so you can run:

```bash
npx @angular/cli@latest new web --directory . --routing --style=scss
```

Then keep/merge the `src/app/features` module layout already defined below.

## Planned UI capabilities

- Personalized multi-user dashboards
- Widget drag/drop and resize (`gridstack` recommended)
- Ticker/timeframe configurable analytics cards
- Live stream subscriptions over WebSocket
- Strategy lifecycle controls and alerts
